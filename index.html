All About Olaf

An unofficial mobile app for St. Olaf College.

This iOS and Android app provides valuable information to students, faculty, staff, and visitors of St. Olaf College.

Features
  • Browse cafe menus
  • Access balances and meal plan info
  • Release print jobs
  • Explore student work opportunities
  • Search for courses
  • View building open and close schedules
  • Peruse all campus-wide calendars
  • Access the campus directory and important contacts
  • Tune-in to streaming events and radio programs
  • Keep up with campus news
  • See the campus at a birds-eye view with the map
  • Track the local bus routes
  • Define campus terms with the dictionary
  • Discover clubs and organizations
  • Quick access to Moodle

Class Dumper

Examine Objective-C runtime information stored in Mach-O files.

Features
  • Invoke class-dump 3.5 (64 bit) from the file picker
  • Save results to a local sqlite database with GRDB
  • Manage previous dumps from a finder-like interface

Bottomless

An unofficial mobile app for Bottomless subscribers.

This iOS and iPadOS app provides Bottomless coffee users a native experience for managing their account.

Features
  • Login
  • Listing your orders (up next, in-progress, past)
  • Graphing the scale’s weight and showing the scale’s status
  • Your referral counts and url
  • Searching/filtering of products
  • Your membership details
  • Preferences for changing your ordering strategy
  • Pausing/unpausing automatic ordering
  • Alert settings
